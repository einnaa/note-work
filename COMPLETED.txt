🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

App not showing?
  → Browser cache: Hard refresh (Cmd+Shift+R on Mac)
  → Port issue: Check terminal for error message
  → Vite will suggest alternative port if 5175 in use

Styles not loading?
  → Check src/index.css is imported in main.tsx
  → Restart dev server: Ctrl+C then npm run dev
  → Clear browser cache

TypeScript errors?
  → They're just warnings, won't stop app
  → Check terminal for error details
  → Fix by updating types as needed

Can't see dev server?
  → Make sure it's running: npm run dev
  → Try: http://localhost:5175
  → If port changed, terminal shows new address
